habitacion sencilla

La habitación sencilla está diseñada para ofrecer comodidad y tranquilidad a huéspedes que viajan solos. Cuenta con una cama individual, baño privado, conexión Wi-Fi, televisión y un ambiente acogedor ideal para descansar después de un día de actividades. Es perfecta para estancias cortas o viajes de negocios.

habitacion doble

La habitación doble ofrece un espacio amplio y confortable, ideal para parejas o dos personas. Dispone de una cama doble o dos camas individuales, baño privado, televisión, conexión Wi-Fi y un ambiente moderno pensado para brindar una estancia relajante y cómoda.

penthouse

El penthouse es la opción más exclusiva del hotel, diseñado para brindar una experiencia de lujo y confort. Cuenta con amplios espacios, cama king size, sala privada, baño de lujo, terraza con vista panorámica y servicios premium. Es ideal para huéspedes que buscan una estancia elegante y una experiencia única.
